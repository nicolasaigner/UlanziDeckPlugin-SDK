Para aumentar a eficiência dos criadores e atender a diversas necessidades, a equipe U‑Studio criou o projeto UlanziDeck. Acreditamos que criadores não são apenas consumidores, mas também produtores — por meio da criação e compartilhamento colaborativo podemos alcançar benefícios mútuos. Lançamos o programa de desenvolvimento de plugins UlanziDeck e, com a colaboração de diversos desenvolvedores, muitos plugins ricos em funcionalidades já foram desenvolvidos. Para facilitar ainda mais e incentivar contribuições, publicamos este repositório open source e convidamos todos os desenvolvedores a participar e construir uma plataforma de plugins mais aberta, flexível e poderosa.

---

## 1. Biblioteca de componentes para plugins

A biblioteca open source para desenvolvimento de plugins tem como objetivo oferecer chamadas de protocolo simples e fáceis de usar, despacho de eventos, conexões de rede seguras e estáveis — fornecendo um conjunto de ferramentas, interfaces e convenções para reduzir a curva de entrada no desenvolvimento de plugins. O escopo inclui (mas não se limita a) encapsulamento de protocolo, componentes reutilizáveis, componentes de diálogo personalizados, etc.
